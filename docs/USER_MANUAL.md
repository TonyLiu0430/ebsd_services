# EBSD 系統使用說明

正式網址：<https://ebsd.cloudlinux.win/>

## 1. 系統導覽

登入後，畫面上方會顯示主選單：

- **EBSD**：返回主要頁面。已登入時會回到「檔案庫」，未登入時會前往「登入」。
- **檔案庫**：管理靶材、上傳 EBSD 掃描檔、查看目前帳號已保存的檔案。
- **自動化報表**：選擇分析樣本與 Golden 樣本，產生 EBSD 分析報表。
- **登出**：登出目前帳號並回到登入頁。

未登入使用者進入檔案庫或自動化報表時，系統會自動導向登入頁。

## 2. 登入與註冊

### 2.1 登入

1. 進入「登入」頁面。
2. 輸入使用者名稱與密碼。
3. 點選「登入」。
4. 登入成功後會自動進入「檔案庫」。

![登入頁面](images/user-manual/01-login.png)

### 2.2 建立帳號

1. 在登入頁點選「註冊」。
2. 輸入使用者名稱與密碼。
3. 點選「建立帳號」。
4. 建立成功後會自動登入並進入「檔案庫」。

帳號規則：

- 使用者名稱需為 3 至 30 個字元。
- 使用者名稱只能使用英文、數字或底線。
- 使用者名稱不可包含空白。
- 密碼需為 4 至 100 個字元。
- 密碼不可包含空白。

## 3. 檔案庫

檔案庫用於建立靶材、保存 EBSD 掃描檔，並查看目前帳號底下的檔案狀態。

頁面上方會顯示目前帳號、靶材數量與掃描檔組數。點選「重新整理」可重新讀取最新資料。

![檔案庫總覽](images/user-manual/02-library-overview.png)

### 3.1 靶材管理

靶材區塊會依材料分類顯示靶材清單，每個靶材會顯示已保存的掃描檔組數。

材料分類由系統自動判定。上傳 EBSD 檔案後，系統會讀取 `.cpr` 檔案中的晶格常數，並依晶格常數比對材料類型，例如 `Cu (FCC)`。使用者不需要手動輸入材料名稱。

新增靶材：

1. 在「新增靶材名稱」欄位輸入靶材名稱。
2. 點選「新增」。
3. 新增完成後，該靶材會自動成為目前選擇的靶材。

注意事項：

- 靶材名稱不可空白。
- 已存在的靶材名稱無法重複新增。

### 3.2 位置命名規則

系統使用九宮格位置格式表示掃描位置。

欄位方向：

- **C**：內側 Center。
- **M**：中間 Middle。
- **E**：外側 Edge。

列方向：

- **U**：上方 Up。
- **M**：中間 Middle。
- **B**：下方 Bottom。

標準 9 個位置如下：

| 位置 | 說明 |
| --- | --- |
| C-U | 內側上方 |
| C-M | 內側中間 |
| C-B | 內側下方 |
| M-U | 中間上方 |
| M-M | 中心位置 |
| M-B | 中間下方 |
| E-U | 外側上方 |
| E-M | 外側中間 |
| E-B | 外側下方 |

在單筆上傳時，使用者可直接點選位置圖，也可在「手動位置」輸入位置。保存時系統會統一轉成 `C-U`、`M-M`、`E-B` 這類標準格式。

## 4. 新增位置掃描檔

「新增位置掃描檔」用於一次上傳一組 EBSD 檔案。每組檔案必須包含一個 `.crc` 與一個 `.cpr`。

![新增位置掃描檔](images/user-manual/03-single-upload.png)

操作步驟：

1. 在「選擇靶材」選擇要保存的靶材。
2. 在「版本」選擇既有版本，或選擇「新增版本」並輸入版本編號。
3. 在「位置」區塊點選九宮格位置，或在「手動位置」輸入位置。
4. 分別選擇 `.crc` 與 `.cpr` 檔案。
5. 確認狀態顯示「CRC + CPR 已選齊」。
6. 點選「保存到靶材」。

檔案規則：

- CRC 欄位只能選擇 `.crc` 檔案。
- CPR 欄位只能選擇 `.cpr` 檔案。
- `.crc` 與 `.cpr` 的主檔名需一致，例如 `C-U.crc` 與 `C-U.cpr`。
- 材料分類會依 `.cpr` 檔案中的晶格常數自動判定。
- 位置不可空白，且需符合標準九宮格位置。

其他操作：

- 點選「清除」可清空目前已選檔案與位置。
- 上傳成功後，系統會顯示保存成功訊息，並重新整理檔案庫。

## 5. 資料夾批次匯入

「資料夾批次匯入」可一次匯入多個靶材或多個版本的 EBSD 檔案。

![資料夾批次匯入](images/user-manual/04-folder-import.png)

操作步驟：

1. 點擊「點擊或拖曳 EBSD 資料夾」。
2. 選擇資料夾，或將資料夾拖曳到匯入區。
3. 系統會自動解析資料夾內的 `.crc` 與 `.cpr` 檔案。
4. 確認畫面顯示已解析的靶材數、版本數與檔案組數。
5. 點選「匯入資料夾」。

批次匯入規則：

- 系統會自動配對相同位置的 `.crc` 與 `.cpr`。
- 只有成功配對的檔案組會出現在匯入清單中。
- 每組檔案匯入後，系統會依 `.cpr` 檔案中的晶格常數自動判定材料分類。
- 資料夾名稱結尾為 `-1`、`-2`、`-3` 等數字時，會視為同一靶材的不同版本。
- 資料夾名稱沒有版本數字時，系統會視為第 1 版。

範例：

| 資料夾名稱 | 系統判定 |
| --- | --- |
| SampleA-1 | 靶材 SampleA，第 1 版 |
| SampleA-2 | 靶材 SampleA，第 2 版 |
| SampleB | 靶材 SampleB，第 1 版 |

其他操作：

- 點選「清除資料夾」可清空目前解析結果。
- 若找不到可配對檔案，畫面會顯示「找不到可配對的 .crc/.cpr」。

## 6. 目前帳號檔案

「目前帳號檔案」會列出目前登入帳號底下保存的 EBSD 掃描檔。

![目前帳號檔案](images/user-manual/05-account-files.png)

### 6.1 篩選靶材

使用「靶材篩選」可查看全部靶材，或只查看指定靶材。

### 6.2 靶材狀態

每個靶材卡片會顯示：

- 靶材名稱。
- 已保存位置數。
- 版本數。
- 最後更新時間。
- 標準 9 點是否完整。
- 缺少哪些標準位置。

九宮格狀態會標示各位置是否已有資料；若有標準 9 點之外的位置，會另外顯示在下方。

### 6.3 版本查看

靶材卡片提供版本切換：

- **完整位置**：顯示截至最新版本的完整位置狀態。
- **各版本按鈕**：只查看該版本內包含的位置。
- 最新版本會以樣式標示。

### 6.4 檔案明細

展開「檔案明細」後可查看每組掃描檔的：

- 位置。
- 掃描檔名稱。
- 版本標籤。
- 材料分類。此分類依 `.cpr` 檔案中的晶格常數自動判定。
- CRC 檔名與大小。
- CPR 檔名與大小。

### 6.5 移除與刪除

移除單一掃描檔：

1. 展開「檔案明細」。
2. 點選該掃描檔旁的「移除」。
3. 在確認視窗中確認移除。

刪除整個靶材：

1. 在靶材卡片右上方點選「刪除靶材」。
2. 在確認視窗中確認刪除。

注意：刪除靶材會刪除目前帳號底下該靶材的所有掃描檔。

## 7. 自動化報表

自動化報表用於從檔案庫選擇分析樣本與 Golden 樣本，產生 EBSD 比對報表。

頁面上方會顯示檔案庫中的掃描檔組數與靶材數量。點選「重新讀取」可重新載入檔案庫資料。

目前報表分析請使用銅（Cu）樣本。本手冊報表示範使用 `DATA07` 作為分析樣本、`DATA09` 作為 Golden 樣本；其他材料目前可用於檔案庫保存與檔案管理展示。

![自動化報表頁面](images/user-manual/06-report-setup.png)

### 7.1 選擇報表資料來源

操作步驟：

1. 在「分析樣本」下拉選單選擇要分析的靶材。
2. 在「Golden 樣本」下拉選單選擇基準靶材。
3. 確認兩者不是同一個樣本。
4. 確認兩者為相同金屬。
5. 點選「產生報表」。

也可以在「檔案庫概覽」中快速選取：

- 左鍵點擊靶材卡片：指定為分析樣本。
- 右鍵點擊靶材卡片：指定為 Golden 樣本。

限制與提示：

- 分析樣本與 Golden 樣本必須是相同金屬。
- 若金屬不一致，系統會顯示錯誤訊息，且無法產生報表。
- 畫面會顯示可比對的位置數量。
- 報表會使用各靶材最新版本的完整位置狀態作為資料來源。

![已選擇分析樣本與 Golden 樣本](images/user-manual/07-report-selected.png)

## 8. 報表操作

產生報表後，頁面會顯示報表工具區與多個分析區塊。請先確認畫面上方的分析樣本與 Golden 樣本，再往下查看各區塊。

![報表結果與浮動工具](images/user-manual/08-report-result.png)

### 8.1 Report 版本

報表左側浮動工具會顯示目前分析樣本版本與 Grain size 設定。

![Report 版本與 Grain size 設定](images/user-manual/09-report-version-grain-size.png)

- 可使用滑桿切換分析樣本版本。
- Golden 樣本使用其最新版本。
- 切換版本後，報表中的分析樣本資料會同步更新。
- 若分析樣本只有一個版本，滑桿無法切換，維持目前版本即可。

### 8.2 Grain size 設定

在浮動工具的「Grain size 設定」中可調整 `Min-grain size (px)`。

操作方式：

1. 輸入 Min-grain size 數值。
2. 點選「套用」。
3. 報表會依新設定重新載入 Grain 資料。

注意事項：

- 數值變更後需點選「套用」，畫面才會更新。
- 點選「套用」後，請等待報表重新載入完成。

### 8.3 輸出報表

報表上方提供兩種輸出：

![輸出 PDF 與 PPT](images/user-manual/10-report-export-buttons.png)

- **輸出 PPT**：新版報表格式，建議優先使用。系統會下載 `.pptx` 檔案。
- **輸出 PDF**：產生接近網頁畫面的 PDF 結果，處理時間較長。僅在需要 PDF 版面時使用。

輸出檔名會包含分析樣本名稱與輸出日期。

操作方式：

1. 確認報表已產生。
2. 若有需要，先完成版本、Grain size、IPF 或特徵分析設定。
3. 建議點選「輸出 PPT」下載新版報表。
4. 若需要接近網頁畫面的 PDF 結果，再點選「輸出 PDF」。
5. 等待按鈕狀態結束後，瀏覽器會下載檔案。

注意事項：

- 匯出前請先確認畫面中使用的是正確分析樣本與 Golden 樣本。
- 若剛調整設定，請等圖表更新完成後再輸出。
- 「輸出 PDF」處理時間較長，等待期間請勿重複點擊。
- 若輸出失敗，請重新整理頁面、重新產生報表後再試一次。

## 9. 報表內容說明

### 9.1 Grain 粒徑分布比對

此區塊以九宮格顯示各位置的 Grain 粒徑分布，並同時比較分析樣本與 Golden 樣本。

![Grain 累積曲線模式](images/user-manual/11-grain-cdf-mode.png)

圖形模式：

- **累積曲線**：顯示粒徑累積分布。
- **分布直方圖**：顯示粒徑分布區間。
- **面積加權直方圖**：依 grain 等效直徑換算面積權重後顯示分布。

![Grain 分布直方圖模式](images/user-manual/12-grain-histogram-mode.png)

![Grain 面積加權直方圖模式](images/user-manual/13-grain-area-histogram-mode.png)

切換圖形模式：

1. 進入「Grain 粒徑分布比對」區塊。
2. 點選「累積曲線」、「分布直方圖」或「面積加權直方圖」。
3. 確認九宮格中的每個位置圖表已切換。

圖例：

- 藍色線或藍色柱狀為分析樣本。
- 橘色虛線或橘色柱狀為 Golden 樣本。

直方圖設定：

- 點選「設定」可開啟「直方圖設定」。
- 可設定分箱起點為「從 0 開始」或「從最小值開始」。
- 可設定分箱間距為「比例」或「絕對數值」。
- 點選「套用設定」後更新圖表。

![直方圖設定](images/user-manual/14-grain-histogram-settings.png)

直方圖設定操作：

1. 點選 Grain 區塊右上方「設定」。
2. 在「直方圖分箱起點」選擇「從 0 開始」或「從最小值開始」。
3. 在「直方圖間距大小」選擇「比例」或「絕對數值」。
4. 輸入間距數值。
5. 點選「套用設定」。

操作補充：

- 「比例」會依目前資料範圍計算分箱間距。
- 「絕對數值」會使用固定數值作為分箱間距。

位置狀態：

- 若分析樣本有資料但 Golden 缺少，該位置會標示「缺 Golden」。
- 若 Golden 有資料但分析樣本未上傳，該位置會標示「未上傳」。
- 若某位置缺資料，該位置的比對結果不可作為完整判斷依據。

### 9.2 IPF 晶粒取向分佈圖

此區塊顯示各位置的 IPF map，用於查看晶粒取向分布。使用者可切換分析樣本與 Golden 樣本，確認兩者顏色分布是否相近。

![IPF 分析樣本](images/user-manual/15-ipf-map-analysis.png)

![IPF Golden 樣本](images/user-manual/16-ipf-map-golden.png)

可操作項目：

- 使用「分析樣本」與「Golden 樣本」切換目前顯示對象。
- 點選「IPF 色鍵」查看顏色與晶向的對應。
- 點選「設定」調整 IPF 參考方向向量與 De-noindex threshold。

查看 IPF 色鍵：

1. 移到或點選「IPF 色鍵」。
2. 查看三角圖上的 `[001]`、`[101]`、`[111]` 方向。
3. 對照九宮格 IPF map 的顏色分布。

![IPF 色鍵](images/user-manual/17-ipf-color-key.png)

IPF 設定：

- 可快速選擇 X、Y、Z 方向。
- 可手動輸入 X、Y、Z 向量。
- 套用時系統會自動歸一化向量。
- 零向量不可使用。
- De-noindex threshold 可設定 0 至 100，預設為 50%。

![IPF 設定](images/user-manual/18-ipf-settings-dialog.png)

IPF 設定操作：

1. 點選 IPF 區塊右上方「設定」。
2. 在「快速選擇」選擇 X、Y 或 Z，或在「具體向量」手動輸入 X、Y、Z。
3. 如需調整 no-index 點填補門檻，修改 `Threshold (%)`。
4. 點選「套用設定」。
5. 確認 IPF map 已重新載入。

![IPF X 方向套用後](images/user-manual/19-ipf-map-x-direction.png)

操作補充：

- 切換「分析樣本」與「Golden 樣本」只會改變目前顯示的 IPF map。
- 調整 IPF 設定後，分析樣本與 Golden 樣本會使用同一組設定顯示。

### 9.3 九宮格完整數據

此區塊以九宮格顯示各位置的完整數據，可切換分析樣本與 Golden 樣本。

![九宮格完整數據：分析樣本](images/user-manual/20-nine-grid-analysis.png)

![九宮格完整數據：Golden 樣本](images/user-manual/21-nine-grid-golden.png)

可操作項目：

- 使用「分析樣本」與「Golden 樣本」切換目前顯示對象。

每個位置顯示：

- Grain Size：Mean、Max、P75、Count。
- Orientation Ratio：Misorientation 20° 與 15° 下的 `[001]`、`[011]`、`[111]` 比例。

若該位置沒有資料，會顯示「無數據」。

欄位說明：

- Mean：平均 grain size。
- Max：最大 grain size。
- P75：第 75 百分位 grain size。
- Count：grain 數量。
- Orientation Ratio：不同 misorientation 門檻下的晶向比例。

### 9.4 晶粒取向比例三角圖

報表提供兩個三角圖區塊：

- Misorientation 20°。
- Misorientation 15°。

![晶粒取向比例三角圖](images/user-manual/22-orientation-triangle-all-rows.png)

可操作項目：

- 使用「顯示位置」切換上、中、下位置列是否顯示。
- 使用「數字顯示」右側按鈕切換圖上數值是否顯示。

三角圖控制列位置如下。左側是「顯示位置」，右側是「數字顯示」。

![三角圖控制列：數字顯示開啟](images/user-manual/28-orientation-triangle-controls-labels-visible.png)

![晶粒取向比例三角圖：篩選位置列](images/user-manual/23-orientation-triangle-filtered.png)

點選「數字顯示」旁的「隱藏」後，圖上的數值標籤會消失，按鈕文字會變成「顯示」。

![晶粒取向比例三角圖：數字隱藏](images/user-manual/29-orientation-triangle-labels-hidden.png)

![三角圖控制列：數字顯示關閉](images/user-manual/30-orientation-triangle-controls-labels-hidden.png)

圖例說明：

- 分析樣本以實線表示。
- Golden 樣本以虛線表示。
- 上、中、下位置列以不同顏色表示。

操作方式：

1. 進入「晶粒取向比例」區塊。
2. 查看 20° 或 15° 的三角圖。
3. 點選「上 (U)」、「中 (M)」、「下 (B)」可顯示或隱藏該位置列。
4. 若要隱藏圖上的數字，找到控制列右側「數字顯示」，點選旁邊的「隱藏」。
5. 若要重新顯示數字，再點同一個位置的「顯示」。

圖例：

- 分析樣本以實線表示。
- Golden 樣本以虛線表示。
- 20° 與 15° 為不同 misorientation 門檻，請依需求切換查看。

### 9.6 特徵分析

「特徵分析」可選擇多個分析項目，並以九宮格顯示分析結果。

![特徵分析設定](images/user-manual/26-feature-analysis-settings.png)

操作步驟：

1. 在「選擇分析項目」勾選要分析的項目。
2. 在「顏色範圍設定（%）」調整綠、黃、紅的門檻。
3. 點選「套用分析」。
4. 查看「分析結果」。

分析項目簡易說明：

| 分析項目 | 說明 |
| --- | --- |
| overall distribution : wasserstein_distance / mean | 比較分析樣本與 Golden 樣本的 grain size 整體分布差異，並以 Golden 樣本的平均 grain size 做標準化。 |
| mean | 比較分析樣本與 Golden 樣本的平均 grain size 差異。 |
| max | 比較分析樣本與 Golden 樣本的最大 grain size 差異。 |
| 25 percentiles | 比較分析樣本與 Golden 樣本的第 25 百分位 grain size 差異。 |
| 75 percentiles | 比較分析樣本與 Golden 樣本的第 75 百分位 grain size 差異。 |
| standard deviation | 比較分析樣本與 Golden 樣本的 grain size 標準差差異。 |
| [001] (deviation 20%) | 比較 misorientation 20% 條件下 `[001]` 方向比例與 Golden 樣本的差異。 |
| [110] (deviation 20%) | 比較 misorientation 20% 條件下 `[110]` 方向比例與 Golden 樣本的差異。 |
| [111] (deviation 20%) | 比較 misorientation 20% 條件下 `[111]` 方向比例與 Golden 樣本的差異。 |
| [001] (deviation 15%) | 比較 misorientation 15% 條件下 `[001]` 方向比例與 Golden 樣本的差異。 |
| [110] (deviation 15%) | 比較 misorientation 15% 條件下 `[110]` 方向比例與 Golden 樣本的差異。 |
| [111] (deviation 15%) | 比較 misorientation 15% 條件下 `[111]` 方向比例與 Golden 樣本的差異。 |

系統會針對有共同位置的分析樣本與 Golden 樣本計算所選項目。若同時勾選多個分析項目，系統會彙整所選項目的結果並顯示於九宮格。

![特徵分析結果](images/user-manual/27-feature-analysis-result.png)

注意事項：

- 使用特徵分析前需先產生報表。
- 至少需選擇一個分析項目。
- 需選擇 Golden 樣本。
- 若分析樣本與 Golden 樣本沒有共同位置，系統會提示無法進行比較。
- 若部分位置缺少 Golden 對應資料，系統會提示缺少的位置，該位置不納入分析。

顏色說明：

- 綠色、黃色、紅色依「顏色範圍設定（%）」的門檻顯示。
- 可用滑桿調整三種顏色的範圍。

操作補充：

- 至少需勾選一個分析項目，才能點選「套用分析」取得結果。
- 可同時勾選多個分析項目。
- 顏色門檻可依使用需求調整。

## 10. 操作流程

1. 登入系統。
2. 進入「檔案庫」。
3. 新增靶材，或使用資料夾批次匯入建立靶材與版本。
4. 確認各靶材標準 9 點資料是否完整。
5. 進入「自動化報表」。
6. 選擇分析樣本與 Golden 樣本。
7. 點選「產生報表」。
8. 視需求切換版本、調整 Grain size、IPF 或直方圖設定。
9. 查看 Grain、IPF、九宮格數據、取向圖與特徵分析結果。
10. 點選「輸出 PPT」保存新版報表；如需接近網頁畫面的 PDF 結果，再點選「輸出 PDF」。
